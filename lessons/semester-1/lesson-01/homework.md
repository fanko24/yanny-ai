# 第一课：课后作业

## 作业1：画一幅"我眼中的AI" ⭐⭐⭐

**要求**：
- 用你喜欢的颜色和方式画画
- 画一画你想象中的AI是什么样子的
- 可以在画旁边写上几句话，说明画的是什么

**提示**：
- AI可以是任何样子哦！人形、动物形、或者你喜欢的任何形状
- 它可能在做什么？帮助人？玩游戏？做作业？
- 用你最亮的颜色！

---

## 作业2：找出家里的3个AI产品 ⭐⭐⭐

**要求**：
- 在家里找找看，找出3个使用人工智能的产品
- 把它们的名字和用途写下来

**记录表**：

| 产品名称 | 它能做什么？ | 我用它来做什么？ |
|---------|-------------|----------------|
| 1.      |             |                 |
| 2.      |             |                 |
| 3.      |             |                 |

**提示**：
- 可以问问爸爸妈妈，哪些是AI产品
- 想想你的手机、家里的电视、智能音箱...
- 记不清楚的话可以画图代替写字

---

## 作业3：和AI说句话 ⭐

**要求**：
- 找一个智能音箱或手机上的语音助手
- 对它说一句你想说的话
- 看看它怎么回答你

**可以问**：
- "你叫什么名字？"
- "你会做什么？"
- "你能帮我吗？"

**想一想**：
- AI回答得怎么样？
- 它好像真的听懂了你说的话吗？

---

## ✨ 作业完成检查

完成作业后，可以让孩子自己检查：

- [ ] 画了一幅AI的画
- [ ] 找到了3个AI产品
- [ ] 和AI说了一句话
- [ ] （可选）给爸爸妈妈讲讲今天学到了什么

---

## 家长提示

1. **鼓励为主**: 无论画得怎样、写得怎样，都要表扬孩子认真完成了作业
2. **协助记录**: 如果孩子写字困难，可以帮忙记录，让孩子口述
3. **分享作品**: 作业完成后，可以让孩子分享给家人，增加成就感
4. **保存作品**: 把孩子的画和记录保存下来，之后可以看到进步

---

## 作业提交

- 完成作业后，拍照记录
- 下次课前检查作业
- 可以和孩子一起回顾，加深印象
