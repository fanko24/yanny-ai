# 第一课：什么是人工智能？

> 和孩子一起探索神奇的人工智能世界

## 课程概览

| 项目 | 内容 |
|------|------|
| 主题 | 什么是人工智能？ |
| 时长 | 45分钟 |
| 对象 | 8岁儿童 |
| 所需材料 | 纸笔、智能设备（手机/音箱）、家中电器 |

## 课程目标

- 了解人工智能的基本概念
- 认识生活中的人工智能应用
- 激发对AI学习的好奇心

## 课程文件

- [课程内容](content.md) - 详细的知识讲解
- [动手实践](hands-on.md) - 互动活动和实验
- [课后作业](homework.md) - 巩固练习
- [家长指导](parent-guide.md) - 授课要点

## 时间安排

| 环节 | 时间 |
|------|------|
| 趣味开场 | 5分钟 |
| 知识讲解 | 20分钟 |
| 互动讨论 | 10分钟 |
| 动手实践 | 8分钟 |
| 总结回顾 | 2分钟 |

## 课程亮点

- 用"聪明的机器人"比喻讲解AI
- 通过生活中的例子让孩子易于理解
- 互动问答激发孩子思考
