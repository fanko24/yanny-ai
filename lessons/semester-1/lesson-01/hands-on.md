# 第一课：动手实践

## 活动1：找一找家里的AI（5分钟）

### 任务
和孩子一起在房间里走一圈，找出使用人工智能的设备。

### 提示

可以找找这些物品：
- 智能音箱
- 智能电视
- 手机/平板
- 扫地机器人
- 智能手表
- 带人脸识别的设备

### 记录表

| 找到的设备 | 它怎么用到了AI？ |
|-----------|-----------------|
|           |                 |
|           |                 |
|           |                 |
|           |                 |

### 小互动
让孩子选一个最喜欢的AI设备，说说为什么喜欢它。

---

## 活动2：和AI对话（3分钟）

### 任务
和孩子一起测试智能语音助手。

### 可以问的问题
- "今天天气怎么样？"
- "给我讲个故事"
- "明天是几号？"
- "唱首歌吧"

### 讨论
- AI回答得怎么样？
- 它有没有答错的时候？
- 你觉得AI听得懂你说的话吗？

---

## 活动3：画一画心中的AI（完成课后作业的一部分）

### 任务
准备纸笔，让孩子画一幅"我眼中的AI"的画。

### 引导问题
- AI长什么样？是人还是机器人？
- 它在做什么？
- 它有什么特别的地方？

完成后可以贴在墙上，记录下孩子第一次对AI的想象！

---

## 🌟 实践小贴士

1. **多提问**: 在活动中多问孩子"为什么"，鼓励他们思考
2. **不限制**: 孩子的答案可能很天马行空，不要否定他们的想法
3. **多观察**: 引导孩子仔细观察生活中的科技产品
4. **拍照记录**: 用手机拍下孩子找到的AI设备，之后可以复习
