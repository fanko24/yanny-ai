# 第12课：成果展示会

## 课程信息
- **模块**：模块三 - AI学习伙伴
- **时长**：40分钟
- **核心体验**：展示学习成果，庆祝学习完成

## 学习目标
1. 学会展示自己的作品
2. 能够讲述学习故事
3. 总结学习收获

## 准备工作
- [ ] 平板电脑或电脑
- [ ] AI工具账号（豆包或ChatGPT）（已注册登录）
- [ ] 作品集和所有作品
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 开场庆祝 | 5分钟 | 庆祝学习完成 |
| 成果展示 | 25分钟 | 展示所有作品 |
| 学习总结 | 5分钟 | 总结学习收获 |
| 颁奖环节 | 5分钟 | 颁发学习证书 |

## 学习准备清单

### 材料准备
- ✓ 设备电量充足
- ✓ 网络连接正常
- ✓ AI工具账号（豆包或ChatGPT）已登录
- ✓ 所有作品和作品集

### 心理准备
- ✓ 自信展示
- ✓ 为自己自豪
- ✓ 享受展示

### 家长准备
- ✓ 陪伴展示
- ✓ 认真聆听
- ✓ 给予鼓励

## 学习小贴士

1. 自信地展示
2. 大胆地讲述
3. 为自己自豪
4. 享受展示过程
