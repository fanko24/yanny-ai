# 第9课：作业小帮手

## 课程信息
- **模块**：模块三 - AI学习伙伴
- **时长**：40分钟
- **核心体验**：用AI辅助学习，合理使用AI工具

## 学习目标
1. 学会合理使用AI辅助学习
2. 能用AI解决学习难题
3. 培养良好的学习习惯

## 准备工作
- [ ] 平板电脑或电脑
- [ ] 豆包账号（已注册登录）
- [ ] 学习用品
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 故事开场 | 5分钟 | 学习伙伴小智的故事 |
| 动手实践 | 25分钟 | 学习使用AI辅助学习 |
| 知识讲解 | 5分钟 | 了解AI学习助手 |
| 创意任务 | 5分钟 | 设计学习计划 |

## 学习准备清单

### 材料准备
- ✓ 设备电量充足
- ✓ 网络连接正常
- ✓ 豆包账号已登录
- ✓ 学习用品

### 心理准备
- ✓ 认真学习态度
- ✓ 好奇心
- ✓ 耐心思考

### 家长准备
- ✓ 陪伴指导
- ✓ 监督使用
- ✓ 记录进步

## 学习小贴士

1. 先自己思考，再问AI
2. 学会验证AI的回答
3. 把AI当作辅助工具
4. 培养独立思考能力
