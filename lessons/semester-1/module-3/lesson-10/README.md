# 第10课：科学探险家

## 课程信息
- **模块**：模块三 - AI学习伙伴
- **时长**：45分钟
- **核心体验**：探索科学世界，培养科学思维

## 学习目标
1. 学会用AI探索科学知识
2. 培养科学探究精神
3. 理解基本的科学现象

## 准备工作
- [ ] 平板电脑或电脑
- [ ] ChatGPT账号（已注册登录）
- [ ] 科学实验材料（简单）
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 故事开场 | 5分钟 | 科学博士小问号的故事 |
| 动手实践 | 30分钟 | 科学探索实践 |
| 知识讲解 | 5分钟 | 了解科学探索 |
| 创意任务 | 5分钟 | 设计科学计划 |

## 学习准备清单

### 材料准备
- ✓ 设备电量充足
- ✓ 网络连接正常
- ✓ ChatGPT账号已登录
- ✓ 简单实验材料

### 心理准备
- ✓ 保持好奇心
- ✓ 大胆提问
- ✓ 耐心探索

### 家长准备
- ✓ 陪伴指导
- ✓ 准备实验材料
- ✓ 记录探索过程

## 学习小贴士

1. 多问为什么
2. 动手做实验
3. 记录发现
4. 分享成果
