# 课后任务：我的科学探索报告

## 任务说明
完成这节课的作业，你需要进行一项科学探索，并制作一份完整的科学探索报告，记录你的发现和收获！

## 任务卡片

### 任务1：科学探索日记（必做）
**创作主题**：记录一次完整的科学探索过程

**探索项目选择**：
选择一个你感兴趣的探索主题：

❓ 植物生长观察
- 种子发芽观察
- 植物向光性实验
- 不同植物的对比

❓ 动物行为观察
- 蚂蚁搬家观察
- 小鸟筑巢观察
- 鱼的游泳观察

❓ 自然现象探索
- 天气变化记录
- 月亮圆缺观察
- 影子变化观察

❓ 科学小实验
- 水的浮沉实验
- 光的折射实验
- 声音传播实验

**日记格式要求**：
```
科学探索日记

探索主题：_________________________
探索时间：____年__月__日 至 ____年__月__日
探索地点：_________________________
天气情况：_________________________

第一天：
观察：______________________________________
发现：______________________________________
疑问：______________________________________

第二天：
观察：______________________________________
发现：______________________________________
疑问：______________________________________

（继续记录...）

总结：
1. 我学到了什么？
   _________________________________________
2. 最有趣的现象是什么？
   _________________________________________
3. 我还有什么疑问？
   _________________________________________
```

**示例日记**：
```
科学探索日记 - 绿豆发芽观察

探索主题：绿豆发芽需要什么？
探索时间：2024年3月10日 - 3月15日
探索地点：窗台
天气情况：晴天

第一天（3月10日）：
观察：绿豆是绿色的，小小的，硬硬的
发现：绿豆没有变化
疑问：绿豆什么时候才会发芽？

第二天（3月11日）：
观察：绿豆变胖了一点，好像裂开了
发现：绿豆吸水了
疑问：为什么会变胖？

第三天（3月12日）：
观察：绿豆裂开了，露出白色的小芽
发现：绿豆开始发芽了！
疑问：小芽会怎么长？

...

总结：
1. 我学到了：绿豆需要水才能发芽
2. 最有趣的是：绿豆会先长根，再长芽
3. 我的疑问：为什么芽是朝上长的？
```

**示例报告**：
```
科学实验报告

实验名称：彩虹制造实验
实验者：Vivian 班级：一年级
实验日期：2024年3月12日

【实验目的】
我想研究：阳光能不能变成彩虹？
我想知道：彩虹是怎么形成的？

【实验材料】
1. 一个三棱镜
2. 一张白纸
3. 阳光

【实验步骤】
1. 把三棱镜放在阳光下
2. 调整三棱镜的角度
3. 在白纸上观察变化

【实验现象】
我看到了：墙上出现了彩色光带
记录下来：红、橙、黄、绿、蓝、靛、紫

【实验结论】
我发现：阳光透过三棱镜会变成彩虹
这说明了：阳光是由多种颜色的光组成的

【我的思考】
为什么会有这种现象？
因为阳光经过折射分散成了不同颜色的光。
下次我想试试：用水盆和镜子做彩虹。
```

## 作品展示方式

### 方式一：实体作品
1. 用日记本手写记录
2. 画上插图或贴照片
3. 制作封面和封底
4. 在班级展示

### 方式二：电子报告
1. 用电脑制作文档
2. 插入图片和视频
3. 添加动画效果
4. 保存为PDF

### 方式三：视频展示
1. 录制探索过程视频
