## 作品整理练习

### 练习1：收集作品
**步骤**：
1. 列出所有课程
2. 找到每课的作品
3. 按课次整理
4. 确保没有遗漏

**作品清单模板**：
```

作品：对话记录
文件位置：__________
完成日期：____年__月__日

第2课：AI画梦工厂
作品：AI画作
文件位置：__________
完成日期：____年__月__日

...（继续记录）
```

### 练习2：作品分类
**分类方法**：
- 按课程分类
- 按类型分类
- 按时间分类
- 按主题分类

**分类示例**：
❓ 按课程分类
- 模块一：认识AI
- 模块二：AI创作
- 模块三：AI学习

❓ 按类型分类
- 对话类
- 绘画类
- 故事类
- 音乐类
- 配音类
- 学习类

❓ 按时间分类
- 第一周（课1-2）
- 第二周（课3-4）
- 第三周（课5-6）
- 第四周（课7-8）
- 第五周（课9-10）

### 练习3：作品说明
**为每件作品添加说明**：
- 作品名称
- 创作时间
- 创作灵感
- 作品描述
- 个人感受

**作品说明模板**：
```
作品信息卡

作品名称：_________________________
所属课程：_________________________
创作时间：____年__月__日

创作灵感：
___________________________________________

作品描述：
___________________________________________

个人感受：
___________________________________________
```

**示例作品说明**：
```
作品信息卡

作品名称：会飞的小猫
所属课程：第2课 AI画梦工厂
创作时间：2024年3月8日

创作灵感：
我想，如果小猫有翅膀，一定很可爱！于是就问AI画了一只会飞的猫。

作品描述：
一只可爱的小白猫，长着彩虹色的翅膀，在蓝色的天空中飞翔。小猫的尾巴像棉花糖一样蓬松，眼睛亮晶晶的。

个人感受：
看到画出来的效果，我开心得跳起来！原来可以让AI帮我画我想象的东西，太神奇了！
```

### 练习4：制作作品集
**作品集结构**：

**封面**
- 标题：我的AI作品集

- 作者姓名
- 完成时间

**目录**
1. 我的旅程：学习回顾
2. 我的作品：作品展示
3. 我的收获：成长记录
4. 我的未来：展望与期待

**正文部分**

- 包含课程介绍
- 展示课程作品
- 记录学习收获
- 写下个人感受

**结语**
- 总结学习收获
- 感谢帮助过的人
- 展望未来学习

## 作品集展示方式

### 方式一：纸质作品集
1. 打印整理好的内容
2. 用文件夹装订
3. 设计精美封面
4. 手写或打印

### 方式二：电子作品集
1. 用电脑制作文档
2. 插入图片和音频
3. 制作成PDF
4. 保存到云端

### 方式三：在线作品集
1. 制作网页展示
2. 上传到网站
3. 分享链接
4. 互动交流
