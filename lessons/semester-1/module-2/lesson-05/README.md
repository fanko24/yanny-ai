# 第5课：创意画室

## 课程信息
- **模块**：模块二 - AI创作工坊
- **时长**：45分钟
- **核心体验**：创意绘画创作，发挥想象力

## 学习目标
1. 学会用AI进行创意绘画
2. 能表达独特的艺术想法
3. 理解创意绘画的技巧

## 准备工作
- [ ] 平板电脑或电脑
- [ ] AI工具账号（豆包或ChatGPT，已注册登录）
- [ ] 绘画工具
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 故事开场 | 5分钟 | 彩虹画仙子的故事 |
| 动手实践 | 30分钟 | 创意绘画创作 |
| 知识讲解 | 5分钟 | 了解创意绘画 |
| 创意任务 | 5分钟 | 设计创作计划 |

## 学习准备清单

### 材料准备
- ✓ 设备电量充足
- ✓ 网络连接正常
- ✓ AI工具账号已登录
- ✓ 绘画工具

### 心理准备
- ✓ 发挥想象力
- ✓ 大胆创新
- ✓ 耐心创作

### 家长准备
- ✓ 陪伴学习
- ✓ 协助构思
- ✓ 记录作品

## 学习小贴士

1. 打破常规思维
2. 添加意想不到的元素
3. 混合不同风格
4. 创造独特的视觉效果
