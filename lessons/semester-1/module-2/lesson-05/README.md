# 第5课：小小作家

## 课程信息
- **模块**：模块二 - AI创作工坊
- **时长**：45分钟
- **核心体验**：编写完整故事，体验作家创作过程

## 学习目标
1. 学会独立编写简单故事
2. 能设计故事情节和角色
3. 掌握故事写作的基本技巧

## 准备工作
- [ ] 平板电脑或电脑
- [ ] 豆包账号（已注册登录）
- [ ] 作文本或纸张
- [ ] 彩笔
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 故事开场 | 5分钟 | 文字魔法师的故事 |
| 动手实践 | 30分钟 | 学习故事写作 |
| 知识讲解 | 5分钟 | 了解写作技巧 |
| 创意任务 | 5分钟 | 设计写作计划 |

## 学习准备清单

### 材料准备
- ✓ 设备电量充足
- ✓ 网络连接正常
- ✓ 豆包账号已登录
- ✓ 作文本和彩笔

### 心理准备
- ✓ 保持专注力
- ✓ 大胆表达
- ✓ 耐心写作

### 家长准备
- ✓ 陪伴学习
- ✓ 协助构思
- ✓ 记录写作过程

## 学习小贴士

1. 先想好故事框架
2. 用简单的句子表达
3. 添加有趣的细节
4. 写完后大声朗读检查

让我们一起成为小小作家吧！📝
