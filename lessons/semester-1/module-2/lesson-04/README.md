# 第4课：小小创作家

## 课程信息
- **模块**：模块二 - AI创作工坊
- **时长**：60分钟
- **核心体验**：用AI创作故事，学习写作技巧，体验创作乐趣

## 学习目标
1. 学会用AI续写和创作故事
2. 能构思简单的故事情节和角色
3. 理解故事创作的基本要素（开头、中间、结尾）
4. 掌握故事写作的基本技巧

## 准备工作
- [ ] 平板电脑或电脑
- [ ] ChatGPT账号（已注册登录）
- [ ] 作文本或纸张
- [ ] 纸和彩笔
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 故事开场 | 5分钟 | 故事王国与文字魔法 |
| 动手实践 | 40分钟 | 创作故事+学习写作 |
| 知识讲解 | 10分钟 | 了解故事创作与写作技巧 |
| 创意任务 | 5分钟 | 设计故事角色和写作计划 |

## 学习准备清单

### 材料准备
- ✓ 设备电量充足
- ✓ 网络连接正常
- ✓ ChatGPT账号已登录
- ✓ 作文本和彩笔
- ✓ 绘画工具

### 心理准备
- ✓ 发挥想象力
- ✓ 大胆创作
- ✓ 保持专注力
- ✓ 耐心写作

### 家长准备
- ✓ 陪伴学习
- ✓ 协助构思
- ✓ 记录创作过程
- ✓ 记录写作过程

## 学习小贴士

1. 先想好故事框架和开头
2. 用简单的句子表达
3. 添加有趣的细节和对话
4. 故事要有完整的结构
5. 写完后大声朗读检查
