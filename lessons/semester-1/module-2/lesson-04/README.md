# 第4课：故事动画师

## 课程信息
- **模块**：模块二 - AI创作工坊
- **时长**：50分钟
- **核心体验**：用AI创作动画故事，感受创作乐趣

## 学习目标
1. 学会用AI续写和创作故事
2. 能构思简单的故事情节
3. 理解故事创作的基本要素

## 准备工作
- [ ] 平板电脑或电脑
- [ ] 豆包账号（已注册登录）
- [ ] 纸和彩笔
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 故事开场 | 5分钟 | 故事王国的小精灵 |
| 动手实践 | 30分钟 | 创作动画故事 |
| 知识讲解 | 10分钟 | 了解故事创作 |
| 创意任务 | 5分钟 | 设计故事角色 |

## 学习准备清单

### 材料准备
- ✓ 设备电量充足
- ✓ 网络连接正常
- ✓ 豆包账号已登录
- ✓ 绘画工具

### 心理准备
- ✓ 发挥想象力
- ✓ 大胆创作
- ✓ 耐心等待

### 家长准备
- ✓ 陪伴学习
- ✓ 协助构思
- ✓ 记录创作过程

## 学习小贴士

1. 先想好故事开头
2. 添加有趣的情节
3. 设计可爱的角色
4. 故事要有完整的结构
