# 课后任务：我的创作作品集

## 任务说明
完成这节课的作业，你需要创作2个完整的作品（一个故事+一篇作文），并制作成一本精美的创作作品集！

## 任务卡片

### 任务1：故事创作（必做）
**创作主题**：创造你的专属故事

**创作步骤**：
1. **设计主角**
   - 选择一个动物作为主角
   - 给主角起个可爱的名字
   - 描述主角的性格特点
   - 画下主角的形象

2. **构思情节**
   - 主角遇到了什么问题？
   - 它怎么解决的？
   - 遇到了哪些朋友？
   - 最后怎么样了？

3. **创作故事**
   - 开头："从前有一个[主角名字]..."
   - 中间：描述遇到的困难
   - 结尾：问题解决，皆大欢喜

**示例故事**：
《勇敢的小猫咪咪》
从前有一只叫咪咪的小猫咪，它很勇敢但有点怕高。
一天，小松鼠的风筝卡在了树上，咪咪虽然害怕，但还是勇敢地爬上去帮小松鼠拿回了风筝。
从那以后，咪咪变得更勇敢了，再也不怕高了。

### 任务2：作文创作（必做）
**创作主题**：记录生活中的美好

**创作要求**：
1. 选择一件真实发生的事情
2. 按照"开头-中间-结尾"的结构写
3. 写出具体细节和感受
4. 用上今天学的写作技巧

**故事参考主题**：
- 《难忘的一天》
- 《我的第一次...》
- 《和...在一起》
- 《学到的...》

**写作步骤**：
1. **回忆事件**
   - 想想发生了什么事
   - 当时的心情怎样
   - 有哪些重要细节

2. **整理结构**
   - 开头：什么时候、在哪里、发生了什么
   - 中间：怎么发展的、遇到了谁
   - 结尾：结果怎样、有什么收获

3. **开始写作**
   - 一句话一句话写
   - 写清楚、写具体
   - 加入自己的感受

**示例作文**：
《学骑自行车》
星期六，爸爸教我学骑自行车。爸爸扶着车子，我小心翼翼地骑上去。刚开始我很害怕，总是摔跤。爸爸说："别怕，我在后面扶着。"我鼓起勇气，慢慢蹬脚踏板。突然，我发现爸爸松手了！我居然自己骑起来了！我开心地大喊："我会骑了！"从那以后，我每天都练习，现在我已经能很熟练地骑车了。我明白了，只要勇敢尝试，没有什么是学不会的！

### 任务3：想象故事（选做加分）
**创作主题**：放飞想象的翅膀

**创作要求**：
1. 发挥想象力，创作虚构故事
2. 可以加入魔法、会说话的动物等元素
3. 故事要完整，有开头、中间、结尾

**故事类型选择**：

❓ 魔法故事
- 《神奇的...》
- 《如果我会...》
- 《魔法的一天》

**创作建议**：
- 设计神奇的能力或物品
- 展开有趣的冒险
- 最后回归现实，收获成长

**示例**：
《会飞的铅笔盒》
我的铅笔盒突然会飞了！它带着我飞出窗外，飞到天上。我们看到了云朵做的棉花糖，星星一闪一闪地眨眼睛。铅笔盒告诉我："带你去个好地方！"我们飞到了一座彩虹桥上，桥那边是文具王国。所有的文具都在开会，它们感谢我平时好好爱护它们。铅笔盒说："只要你继续爱护文具，我就会一直陪你飞！"我抱着铅笔盒开心地回家，这真是神奇的一天！

❓ 动物朋友
- 《和...做朋友》
- 《动物...》
- 《森林里的...》

**创作建议**：
- 选择喜欢的动物
- 设计特殊的友谊
- 一起完成某件事

❓ 梦想世界
- 《我梦想的...》
- 《未来的...》
- 《假如我是...》

**创作建议**：
- 描述理想中的场景
- 展开有趣的想象
- 表达美好的愿望

## 制作创作作品集

### 封面设计
- 作品集名称：《____的创作魔法书》
- 设计一个漂亮的封面
- 画上你创作的所有主角
- 写上作者名字和创作日期

### 故事页面
每页的内容：
1. **故事标题**
2. **主角介绍**（配图）
3. **完整故事**
4. **故事插图**（AI生成的图片）
5. **创作心得**（写下创作时的想法）

### 作文页面
每页的内容：
1. **作品标题**
2. **创作日期**
3. **完整作文**
4. **创作说明**（写下创作时的想法）
5. **插画装饰**（自己画或用AI生成）

### 特殊设计
- **角色页**：介绍故事中的所有角色
- **场景页**：描述故事发生的美丽场景
- **对话页**：展示精彩的对话片段
- **作家自画像**：画出你作为作家的样子
- **创作日记**：记录写作过程中的感受

## 创意制作方式

### 方式一：手工创作书
1. 用A4纸折成书本
2. 手写故事内容
3. 画插图或贴贴纸
4. 装订成册，做硬封面

### 方式二：电子作品集
1. 用PPT或文档制作
2. 插入AI生成的图片
3. 添加动画效果
4. 保存为PDF或视频

### 方式三：立体创作书
1. 制作立体场景
2. 角色可以移动
3. 推拉机关展示情节
4. 翻页式故事讲述
