# 第8课：视频魔法师

## 课程信息
- **模块**：模块二 - AI创作工坊
- **时长**：50分钟
- **核心体验**：用AI生成视频，让画面动起来

## 学习目标
1. 了解什么是文生视频技术
2. 学会用AI工具生成短视频
3. 能设计简单的视频创意
4. 体验AI创作的神奇魅力

## 准备工作
- [ ] 平板电脑或电脑
- [ ] AI工具账号（豆包或ChatGPT）（用于生成视频创意）
- [ ] 文生视频工具（如可灵AI、海螺AI等）
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 故事开场 | 5分钟 | 视频魔法师的故事 |
| 动手实践 | 30分钟 | 学习生成视频 |
| 知识讲解 | 10分钟 | 了解文生视频 |
| 创意任务 | 5分钟 | 设计视频创意 |

## 学习准备清单

### 材料准备
- ✓ 设备电量充足
- ✓ 网络连接正常
- ✓ 账号已登录
- ✓ 准备好创意想法

### 心理准备
- ✓ 发挥想象力
- ✓ 大胆尝试
- ✓ 耐心等待生成

### 家长准备
- ✓ 陪伴学习
- ✓ 协助操作
- ✓ 记录创作过程

## 学习小贴士

1. 先想好视频内容
2. 用简单的话描述画面
3. 可以多尝试几次
4. 保存喜欢的作品
