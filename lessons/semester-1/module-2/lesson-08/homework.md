# 课后任务：我的视频作品集

## 任务说明
完成这节课的作业，你需要创作3-5个AI视频，并制作成一个精美的视频作品集！

## 任务卡片

### 任务1：动物视频（必做）
**创作主题**：可爱的小动物

**创作要求**：
1. 选择一个你喜欢的动物作为主角
2. 描述动物的样子和它正在做的事情
3. 生成15-30秒的视频
4. 保存并记录你的描述

**示例**：
- 《小猫追蝴蝶》
- 《小狗在公园奔跑》
- 《小兔子吃胡萝卜》

**描述模板**：
"一只[颜色]的[动物]，[特点描述]，在[地点][做什么动作]"

**创作步骤**：
1. 选择动物：猫、狗、兔子、鸟等
2. 想象场景：草地、森林、公园等
3. 设计动作：跑、跳、飞、游等
4. 写下描述
5. 生成视频
6. 检查效果，不满意就调整再试

---

### 任务2：风景视频（必做）
**创作主题**：美丽的自然风景

**创作要求**：
1. 选择一个自然场景
2. 描述场景中的动态元素
3. 生成15-30秒的视频
4. 保存并记录你的描述

**示例**：
- 《日出》
- 《樱花飘落》
- 《雪花飞舞》

**描述模板**：
"[时间/天气]，[场景描述]，[什么在动/变化]"

**创作步骤**：
1. 选择场景：日出、花海、雪景等
2. 添加颜色：金色、粉色、白色等
3. 描述动态：升起、飘落、飞舞等
4. 写下描述
5. 生成视频

## 制作视频作品集

### 封面设计
- 作品集名称：《____的视频魔法作品集》
- 设计一个漂亮的封面图
- 写上作者名字和创作日期

### 视频页面
每个视频记录：
1. **视频标题**
2. **创作日期**
3. **描述内容**（你输入的魔法咒语）
4. **视频文件**（保存的视频）
5. **创作心得**（写写创作时的感受）

### 作品介绍
- **我最喜欢的视频**：为什么喜欢？
- **最有趣的创作过程**：发生了什么有趣的事？
- **我想改进的地方**：下次想怎么做更好？
- **我的创意来源**：想法从哪里来的？

### 特殊设计
- **魔法咒语本**：记录所有成功的描述
- **创意灵感页**：记录还想创作的视频
- **我的进步**：对比第一次和最后一次的作品

## 创意制作方式

### 方式一：电子相册
1. 把视频和描述整理成PPT
2. 每页一个视频
3. 加上漂亮的背景和装饰
4. 可以配上音乐

### 方式二：视频合集
1. 用视频编辑软件
2. 把所有视频拼接起来
3. 加上标题和过渡效果
4. 配上背景音乐
5. 导出成一个完整的视频

### 方式三：网页展示
1. 创建一个简单的网页
2. 嵌入所有视频
3. 加上描述和心得
4. 分享给朋友和家人

## 作品展示

### 家庭展示
- 举办家庭视频首映会
- 用电视或投影仪播放
- 家人一起观看和点评

### 班级展示
- 在班级分享时间展示
- 讲解创作过程
- 和同学交流心得

### 网络分享（可选）
- 在家长帮助下分享
- 上传到视频平台
- 让更多人看到你的作品
