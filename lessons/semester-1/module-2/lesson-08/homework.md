# 课后任务：我的配音作品

## 任务说明
完成这节课的作业，你需要选择一个故事或动画，为它配音，创作一个有声音的精彩作品！

## 任务卡片

### 任务1：为故事配音（必做）
**创作主题**：选择一个故事，为不同的角色配音

**故事选择**：
选择一个你喜欢的配音主题：

❓ 童话故事
- 三只小猪
- 小红帽
- 白雪公主
- 丑小鸭

❓ 动物故事
- 狼来了
- 龟兔赛跑
- 狐狸和乌鸦
- 拔萝卜

❓ 自己写的故事
- 你创作的故事
- 你编的小故事
- 你想讲的故事
- 任何有趣的故事

❓ 动画片段
- 你喜欢的动画
- 有趣的动画片断
- 绘本故事动画
- 短视频内容

**配音格式要求**：
```
配音作品记录

作品名称：_________________________
配音时间：____年__月__日

故事简介：
___________________________________________

角色列表：
角色1：__________ 声音特点：_____________
角色2：__________ 声音特点：_____________
角色3：__________ 声音特点：_____________

我的配音准备：
☐ 观察角色特点
☐ 确定声音风格
☐ 练习台词
☐ 录制并回听

我的配音感想：
（配音时的心情、遇到的困难、学到了什么）
___________________________________________
___________________________________________
```

**示例配音记录**：
```
配音作品记录

作品名称：三只小猪
配音时间：2024年3月16日

故事简介：
三只小猪离开妈妈，自己盖房子。第一只小猪盖了草房子，第二只小猪盖了木头房子，第三只小猪盖了砖房子。大灰狼来了，吹倒了前两个房子，最后掉进了第三只小猪的热水锅里！

角色列表：
角色1：小猪1  声音特点：活泼可爱，说话快
角色2：小猪2  声音特点：调皮，有点贪心
角色3：小猪3  声音特点：聪明，说话稳重
角色4：大灰狼  声音特点：低沉，有点凶

我的配音准备：
☑ 观察角色特点
☑ 确定声音风格
☑ 练习台词
☑ 录制并回听

我的配音感想：
大灰狼的声音最难配，要发出很低沉的"嗷呜"。小猪3很聪明，我配得很认真。配音的时候要做表情，不然声音就没有感情了。我觉得自己表现得很棒！
```

```
☐ 角色很生动
☐ 还需要改进

___________________________________________

```

**练习计划**：
- 第1-2天：练习声音高低
- 第3-4天：练习情感表达
- 第5-6天：练习角色切换
- 第7天：完整配音一个故事

### 任务4：配音小剧场（选做）
**创作主题**：举办一个小小的配音表演

**剧场准备**：

**1. 选择表演内容**
- 选一个完整的故事
- 确定所有角色
- 准备台词
- 分配角色

**角色分配表**：
```
角色     配音演员    声音特点
_______   ________    ______________
_______   ________    ______________
_______   ________    ______________
_______   ________    ______________
```

**2. 准备表演**
- 练习台词
- 准备道具
- 设计动作
- 确定音效

**3. 邀请观众**
- 邀请家人朋友
- 准备节目单
