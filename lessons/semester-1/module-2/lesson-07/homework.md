# 课后任务：我的创意画展

## 任务说明
完成这节课的作业，你需要创作一幅富有创意的画，并举办一个小小的创意画展，展示你的想象力和创造力！

## 任务卡片

### 任务1：创意绘画（必做）
**创作主题**：发挥想象力，画一幅最独特的创意画

**绘画灵感选择**：
选择一个你喜欢的创作主题：

❓ 万物融合
- 食物动物融合（鸡蛋鸡、面包兔）
- 自然建筑融合（树房子、水城堡）
- 动物植物融合（花蝴蝶、叶兔子）
- 日常神奇融合（椅子鸟、杯火车）

❓ 颠倒世界
- 天空在大地下
- 鱼在天上飞
- 人在云里游泳
- 蚂蚁比大象大

❓ 未来世界
- 会飞的汽车
- 月亮上的城市
- 和机器人做朋友
- 用彩虹桥出行

❓ 迷你世界
- 迷你森林
- 迷你海洋
- 迷你城市
- 迷你你

❓ 神奇动物
- 会飞的鱼
- 长翅膀的猫
- 吃云朵的大象
- 唱歌的树

**绘画格式要求**：
```
创意绘画作品

作品名称：_________________________
创作时间：____年__月__日

我的创意想法：
___________________________________________

我用了什么创意技巧：
☐ 变换颜色
☐ 混合元素
☐ 颠倒规则
☐ 添加意想不到的东西
☐ 创造新事物

我的作品描述：
（描述你画了什么，有什么特别的地方）
___________________________________________
___________________________________________

我的画讲的故事：
（给这幅画编一个小故事）
___________________________________________
___________________________________________
```

**示例作品**：
```
创意绘画作品

作品名称：糖果城市
创作时间：2024年3月15日

我的创意想法：
我想画一个用糖果做的城市，这样就可以随时吃到好吃的！

我用了什么创意技巧：
☑ 变换颜色
☑ 混合元素
☐ 颠倒规则
☐ 添加意想不到的东西
☐ 创造新事物

我的作品描述：
我画了一个神奇的城市，房子是巧克力做的，树是棒棒糖做的，河流里流的是果汁，天上的云朵也是棉花糖做的！所有的东西都可以吃，太棒了！

我的画讲的故事：
在糖果城市里，小朋友们早上醒来，咬一口枕头（软糖），打开窗户（饼干），看到外面下着果汁雨，他们就张开嘴巴喝果汁，甜甜的，真好喝！
```

### 任务3：创意小故事（选做）
**创作主题**：为你的创意画写一个完整的故事

**故事格式要求**：
```
我的创意画故事

作品名称：_________________________

第一章：故事的开始
___________________________________________
___________________________________________

第二章：发生了什么
___________________________________________
___________________________________________

第三章：结局是什么
___________________________________________
___________________________________________

故事想告诉大家什么：
___________________________________________
___________________________________________
```

**故事元素参考**：
- **主角**：你的画里的角色
- **背景**：画里的场景
- **冲突**：发生了什么问题
- **解决**：问题怎么解决
- **主题**：故事想告诉什么道理

**示例故事**：
```
我的创意画故事

作品名称：小猫钓星星

第一章：故事的开始
在一个神奇的小村庄里，住着一只叫咪咪的小猫。咪咪最好的朋友是一条叫小鱼儿的小鱼，它们经常一起玩。

第二章：发生了什么
小鱼儿告诉咪咪，它最大的愿望是看一次真正的星星。但是，小鱼儿住在水里，看不到天上的星星。咪咪心想："我要帮小鱼儿实现愿望！"

第三章：结局是什么
咪咪爬到月亮上，用鱼竿钓下最亮的星星。它把星星装在透明的水瓶里，带给了小鱼儿。小鱼儿在瓶子里看到了美丽的星星，开心极了！

故事想告诉大家什么
真正的朋友会想办法帮助对方实现愿望。只要有爱和勇气，什么困难都能克服！
```

### 任务4：创意画展（选做）
**创作主题**：举办一个小小的创意画展

**画展准备**：

**1. 选择作品**
- 选出你最满意的3-5幅作品
- 给每幅画写一个说明
- 准备画展标题

**作品说明格式**：
```
作品名称：_________________________
创意亮点：_________________________
作品故事：_________________________
```

**2. 布置画展**
- 把画贴在墙上或放在桌子上
- 设计一个画展入口
- 准备参观留言本

**3. 准备讲解**
- 每幅画用1-2句话介绍
- 讲述你的创作想法
- 分享作品背后的故事

**4. 邀请参观**
- 邀请爸爸妈妈
- 邀请好朋友
- 邀请爷爷奶奶外公外婆

**画展海报模板**：
```
========================================
     ✨ Vivian的创意画展 ✨
========================================
时间：____年__月__日
地点：_______________________
作品数量：____幅

参观亮点：
• 独特的创意想法
• 精彩的绘画作品
• 有趣的作品故事

欢迎大家来参观！
========================================
```

## 作品展示方式

### 方式一：实体画展
1. 把画贴在墙上
2. 制作作品说明卡
3. 邀请家人参观
4. 讲解每幅画

### 方式二：电子画册
1. 用电脑整理作品
2. 添加文字说明
3. 制作成PDF
4. 发给朋友看

### 方式三：视频分享
1. 拍摄绘画过程
2. 讲解创意想法
3. 展示作品
4. 分享心得
