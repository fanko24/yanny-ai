# 课后任务：我的音乐魔法盒

## 任务说明
完成这节课的作业，你需要创作2首音乐作品，并制作一个精美的音乐魔法盒！

## 任务卡片

### 任务1：创作儿歌（必做）
**创作主题**：创作一首完整的儿歌

**创作要求**：
1. 选择一个儿歌主题
2. 写出完整的歌词
3. 配上简单的旋律或节奏

**主题参考**：
- 《小动物》系列
  - 《可爱的小猫咪》
  - 《快乐的小狗》
  - 《聪明的小兔子》
  - 《飞翔的小鸟》

- 《自然》系列
  - 《小雨滴》
  - 《小星星》
  - 《小花朵》
  - 《小月亮》

- 《生活》系列
  - 《我的好朋友》
  - 《快乐的一天》
  - 《我爱我的家》
  - 《上学歌》

**创作步骤**：
1. **确定主题**
   - 选择你最喜欢的主题
   - 想想这个主题的特点
   - 确定歌曲的情感（欢快/温柔）

2. **写歌词**
   - 主歌：4-6句，描述内容
   - 副歌：2-4句，重复部分
   - 注意押韵和字数

3. **创作旋律**
   - 用简单的音阶：哆-咪-嗦-啦
   - 找出歌词的节奏
   - 尝试哼唱出来

4. **录制保存**
   - 用手机录下你的歌声
   - 保存音频文件
   - 准备展示

**示例作品**：
《快乐的小猫咪》
主歌：
小猫咪喵喵叫
摇尾巴真可爱
花园里追蝴蝶
阳光下晒太阳
副歌：
喵喵喵 真快乐
蹦蹦跳跳我最棒
我是快乐的小猫猫
整天嘻嘻哈哈笑

### 任务2：音乐故事（选做）
**创作主题**：用音乐讲一个故事

**创作要求**：
1. 选择一个简单的故事
2. 为故事设计不同的音乐段落
3. 表现故事的情感变化

**故事类型参考**：
❓ 动物故事
- 《小兔子找朋友》
- 《小熊的蜂蜜》
- 《小猫的冒险》

❓ 友谊故事
- 《分享的快乐》
- 《互相帮助》
- 《一起成长》

❓ 成长故事
- 《勇敢的心》
- 《不放弃》
- 《学会分享》

**创作步骤**：
1. **选择故事**
   - 想一个简单的3句话故事
   - 确定主角和情节
   - 想好情感变化

2. **分段设计音乐**
   - **开始**：轻柔的音乐，介绍场景
   - **发展**：节奏加快，遇到困难
   - **高潮**：音乐最强烈，解决问题
   - **结局**：音乐变温柔，美好结局

3. **创作各段歌词**
   - 为每段写简单的歌词
   - 配合音乐的情感
   - 保持风格一致

4. **表演录制**
   - 分段表演
   - 讲故事 + 音乐
   - 录制完整作品

**示例**：
《小兔子找朋友》
开始（轻柔）：
"小兔子一个人，好孤单"
音乐：缓慢的钢琴声
发展（稍快）：
"它去找小鸟，小鸟在唱歌"
音乐：加入轻快的节奏
高潮（强）：
"它们一起玩，真快乐"
音乐：欢快的合奏
结局（温柔）：
"交到好朋友，不再孤单"
音乐：渐渐平静

## 制作音乐魔法盒

### 魔法盒设计
- 名称：《____的音乐魔法盒》
- 材料：硬纸盒、彩纸、装饰物
- 设计：音乐主题的装饰

### 魔法盒内容
1. **歌词本**
   - 手写或打印歌词
   - 配上插图
   - 制作精美封面

2. **音频CD**
   - 刻录录制的音乐
   - 设计CD封面
   - 写上曲目信息

3. **音乐卡片**
   - 每首歌一张卡片
   - 写下创作灵感
   - 画上音乐符号

4. **乐器制作**
   - 简单的乐器（沙锤、鼓）
   - 制作说明
   - 演奏方法

### 魔法盒特色
- **音乐开关**：打开盒子播放音乐
- **歌词窗口**：翻开看到歌词
- **惊喜小礼物**：音乐相关小物件

## 创意加分项

### 1. MV制作
- 为歌曲制作简单MV
- 用动画或照片
- 配上你的歌声

### 2. 乐器演奏
- 用简单乐器伴奏
- 如：沙锤、铃铛、鼓
- 录制演奏视频

### 3. 音乐游戏
- 设计音乐猜猜猜游戏
- 音乐传花游戏
- 音乐接龙游戏

## 展示方式
