# 第6课：配音小演员

## 课程信息
- **模块**：模块二 - AI创作工坊
- **时长**：40分钟
- **核心体验**：角色配音表演，体验声音的魅力

## 学习目标
1. 学会为角色配音
2. 能用声音表现不同角色
3. 理解配音的基本技巧

## 准备工作
- [ ] 平板电脑或电脑
- [ ] ChatGPT账号（已注册登录）
- [ ] 录音设备
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 故事开场 | 5分钟 | 声音精灵的故事 |
| 动手实践 | 25分钟 | 学习配音技巧 |
| 知识讲解 | 5分钟 | 了解配音艺术 |
| 创意任务 | 5分钟 | 设计配音计划 |

## 学习准备清单

### 材料准备
- ✓ 设备电量充足
- ✓ 网络连接正常
- ✓ ChatGPT账号已登录
- ✓ 录音设备

### 心理准备
- ✓ 大胆表现
- ✓ 发挥想象力
- ✓ 享受表演

### 家长准备
- ✓ 陪伴学习
- ✓ 协助练习
- ✓ 记录表演

## 学习小贴士

1. 观察角色的特点
2. 用声音表现性格
3. 加入表情和动作
4. 录下来回听改进
