# 第6课：音乐魔法师

## 课程信息
- **模块**：模块二 - AI创作工坊
- **时长**：40分钟
- **核心体验**：创作音乐相关内容，感受音乐的魅力

## 学习目标
1. 学会用AI创作歌词或小曲
2. 能表达音乐的想法和情感
3. 理解音乐的基本要素

## 准备工作
- [ ] 平板电脑或电脑
- [ ] 豆包账号（已注册登录）
- [ ] 录音设备（手机）
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 故事开场 | 5分钟 | 音乐王国的小精灵 |
| 动手实践 | 25分钟 | 创作音乐内容 |
| 知识讲解 | 5分钟 | 了解音乐创作 |
| 创意任务 | 5分钟 | 设计音乐计划 |

## 学习准备清单

### 材料准备
- ✓ 设备电量充足
- ✓ 网络连接正常
- ✓ 豆包账号已登录
- ✓ 录音设备

### 心理准备
- ✓ 喜欢音乐和唱歌
- ✓ 大方表达
- ✓ 保持节奏感

### 家长准备
- ✓ 陪伴学习
- ✓ 协助创作
- ✓ 记录音乐片段

## 学习小贴士

1. 选择自己喜欢的音乐风格
2. 用简单易懂的词语表达
3. 注意歌词的节奏和韵律
4. 可以边创作边哼唱
