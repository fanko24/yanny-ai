# 课后任务：我的声音日记

## 任务说明
完成这节课的作业，你需要和ChatGPT进行一系列有趣的语音互动，并记录下你的声音发现！

## 任务卡片

### 任务1：声音护照（必做）
**创作主题**：用不同声音介绍自己

**创作要求**：
1. 用3种不同的声音和ChatGPT对话：
   - **声音1**：正常声音（你平时的声音）
     "你好，我是[名字]，今年[年龄]岁。"
   - **声音2**：快乐声音（活泼、清脆）
     "大家好！我是快乐的小[名字]！"
   - **声音3**：可爱声音（软软的、甜甜的）
     "呜~我是可爱的小[名字]哦！"

2. 记录AI对每种声音的不同回应

**作业要求**：
- 录下你的3种声音
- 画出3种声音的形象
- 写下哪种声音最让你开心

---

### 任务2：声音动物园（选做）
**创作主题**：模仿动物声音

**创作步骤**：
1. 选择5种你喜欢的动物
2. 用声音模仿每种动物的叫声
3. 让AI猜你模仿的是什么动物

**动物清单**：
❓ 家养动物
- 小狗：汪汪！
- 小猫：喵~
- 小鸡：叽叽叽
- 小鸭：嘎嘎嘎

❓ 野生动物
- 狮子：嗷~~
- 老虎：吼~~
- 猴子：吱吱~
- 大象：呜~~

❓ 鸟类
- 小鸟：叽喳喳
- 布谷鸟：布谷布谷
- 企鹅：哦哦哦
- 孔雀：呜~~

**作业要求**：
- 画出你模仿的动物
- 记录AI猜对了几种
- 写下最有趣的模仿

---

### 任务3：声音故事会（选做）
**创作主题**：合作创作声音故事

**创作步骤**：
1. 你用声音讲一个故事开头
2. AI用声音接下去
3. 一直接下去，创作完整故事

**故事开头建议**：
- "从前有一只小兔子..."
- "在一个美丽的森林里..."
- "有一天，我遇到了..."
- "很久很久以前..."

**作业要求**：
- 记录完整的故事
- 给故事起个题目
- 画出故事中的主要角色
- 标注哪部分是你说的，哪部分是AI说的

## 制作声音日记本

### 封面设计
- 日记本名称：《____的声音魔法日记》
- 设计一个可爱的封面
- 画上声音的符号（音符、喇叭等）
- 写上你的名字和日期

### 内页内容
每页记录：
1. **日期**：____年__月__日
2. **活动名称**：如"声音护照"
3. **声音记录**：
   - 声音描述（"我用快乐的声音唱歌"）
   - AI回应（"ChatGPT说：你的声音真好听！"）
   - 我的感受（"我觉得很开心！"）
4. **声音插画**：画出声音的样子

### 装饰设计
- 用彩笔画音符
- 贴上声音相关的贴纸
- 写上评语："太棒了！"、"真有趣！"
- 给每个声音打分

## 作品展示方式

### 方式一：实体日记本
1. 用笔记本手写记录
2. 画插图装饰
3. 制作目录页
4. 在班级展示

### 方式二：电子日记
1. 用录音APP录下声音
2. 用文字整理内容
3. 插入图片
4. 制作成电子书

### 方式三：声音视频
1. 录制表演视频
2. 展示不同的声音
3. 配上背景音乐
4. 分享给亲友
