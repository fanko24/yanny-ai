# 第3课：AI小歌手

## 课程信息
- **模块**：模块一 - 认识AI朋友
- **时长**：35分钟
- **核心体验**：与AI进行语音互动，体验AI的声音

## 学习目标
1. 学会使用豆包的语音功能
2. 能与AI进行语音对话
3. 理解AI语音交互的原理

## 准备工作
- [ ] 平板电脑或电脑
- [ ] 豆包账号（已注册登录）
- [ ] 耳机（可选）
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 故事开场 | 5分钟 | 声音魔法师的故事 |
| 动手实践 | 20分钟 | 学习语音对话 |
| 知识讲解 | 5分钟 | 了解AI语音 |
| 创意任务 | 5分钟 | 设计声音游戏 |

## 学习准备清单

### 材料准备
- ✓ 设备电量充足
- ✓ 网络连接正常
- ✓ 豆包账号已登录
- ✓ 耳机（可选）

### 心理准备
- ✓ 练习清晰说话
- ✓ 大方表达
- ✓ 保持好奇

### 家长准备
- ✓ 陪伴学习
- ✓ 协助操作
- ✓ 记录有趣对话

## 学习小贴士

1. 说话清晰，声音适中
2. 可以用不同的声音说话
3. 耐心等待AI的回答
4. 保存有趣的语音对话
