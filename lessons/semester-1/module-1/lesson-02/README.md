# 第2课：AI画梦工厂

## 课程信息
- **模块**：模块一 - 认识AI朋友
- **时长**：45分钟
- **核心体验**：用文字生成图片，感受AI绘画的神奇

## 学习目标
1. 学会用文字描述想要的图片
2. 能生成自己喜欢的图画
3. 理解AI绘画的基本原理

## 准备工作
- [ ] 平板电脑或电脑
- [ ] AI工具账号（豆包或ChatGPT）（已注册登录）
- [ ] 彩色画笔和纸
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 故事开场 | 5分钟 | 魔法画笔的故事 |
| 动手实践 | 25分钟 | 学习用文字画画 |
| 知识讲解 | 10分钟 | 了解AI绘画 |
| 创意任务 | 5分钟 | 设计自己的画作 |

## 学习准备清单

### 材料准备
- ✓ 设备电量充足
- ✓ 网络连接正常
- ✓ AI工具账号（豆包或ChatGPT）已登录
- ✓ 画纸和彩笔

### 心理准备
- ✓ 保持想象力
- ✓ 大胆描述
- ✓ 耐心等待

### 家长准备
- ✓ 陪伴学习
- ✓ 协助描述
- ✓ 记录作品

## 学习小贴士

1. 描述要具体，包含颜色、形状
2. 可以添加有趣的细节
3. 多尝试不同的描述
4. 保存喜欢的作品
