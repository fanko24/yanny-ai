# 课后任务：我的AI画册

## 任务说明
完成这节课的作业，你需要和ChatGPT一起创作3幅独特的画作，并制作成一本属于自己的AI画册！

## 任务卡片

### 任务1：自画像（必做）
**创作主题**：我心中的自己

**创作要求**：
1. 先在纸上画出你自己的样子
2. 然后用文字描述给ChatGPT听：
   - "画一个[年龄]岁的[性别]，有[头发颜色]的头发，[眼睛颜色]的眼睛"
   - "穿着[喜欢的衣服]"
   - "[喜欢的表情]"
   - "卡通风格"

**作业要求**：
- 对比你画的画和AI画的画
- 写下3个不同之处
- 写下1个你更喜欢的地方

---

### 任务2：梦幻家园（选做）
**创作主题**：我梦想中的家

**创作要求**：
1. 想象一个特别的家：
   - 可以是树屋、城堡、海底房子等
   - 有特别的功能（会飞、潜水等）
   - 有美丽的装饰

2. 用文字描述给ChatGPT：
   - "画一个[类型]的家"
   - "[特别的功能]"
   - "[周围的景色]"
   - "[喜欢的颜色和风格]"

**作业要求**：
- 给你的房子起个名字
- 写住在里面的感觉
- 画出房子周围的3个好朋友

---

### 任务3：故事插画（选做）
**创作主题**：给故事画插图

**创作步骤**：
1. 先编一个简单的故事（3句话）：
   "从前有一个..."
   "有一天..."
   "最后..."

2. 根据故事内容，让AI画3幅插图：
   - 开头场景
   - 发展场景
   - 结局场景

**作业要求**：
- 把故事写下来
- 把3幅图按顺序排列
- 给故事起个题目

## 制作AI画册

### 封面设计
- 画册名称：《____的AI魔法画册》
- 设计一个漂亮的封面
- 写上你的名字
- 画上你最喜欢的图案

### 内页布局
每页的内容：
1. 画作
2. 创作说明（你说了什么话）
3. 创作心得（你觉得怎么样）
4. 创想延展（如果再画一次会怎样）

### 装饰设计
- 用彩笔给画册装饰边框
- 贴上小贴纸
- 写一些评语："太棒了！"、"真好看！"
- 给每幅画打星星评分

## 作品展示方式

### 方式一：实体画册
1. 用A4纸打印或手绘
2. 装订成册
3. 制作封面和封底
4. 在班级展示

### 方式二：电子画册
1. 用PPT制作
2. 每页放图片和文字
3. 添加动画效果
4. 分享给亲友

### 方式三：画册视频
1. 拍摄画册翻页过程
2. 配上解说
3. 添加背景音乐
4. 制作成小视频

## 创意加分项

### 1. 画册故事
- 把3幅画编成连续故事
- 画出故事中的角色
- 设计故事的发展

### 2. 互动画册
- 制作可以翻页的立体画册
- 添加可以移动的贴纸
- 设计小机关

### 3. 展开画册
- 制作可以展开的大画册
- 画一个全景图
- 添加小细节
