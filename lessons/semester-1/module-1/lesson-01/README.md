# 第1课：和AI聊聊天

## 课程信息
- **模块**：模块一 - 认识AI朋友
- **时长**：40分钟
- **核心体验**：第一次与AI对话，感受AI的智能

## 学习目标
1. 学会使用 ChatGPT 进行对话
2. 理解什么是AI对话
3. 能提出有趣的问题与AI交流

## 准备工作
- [ ] 平板电脑或电脑
- [ ] AI工具账号（推荐豆包，也可以用ChatGPT，已注册登录）
- [ ] 安静的学习环境
- [ ] 家长陪伴

> 💡 推荐使用豆包，也可以用ChatGPT作为替代

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 故事开场 | 5分钟 | 认识新朋友 ChatGPT |
| 动手实践 | 20分钟 | 学习与 ChatGPT 对话 |
| 知识讲解 | 10分钟 | 了解AI对话 |
| 创意任务 | 5分钟 | 想想怎么问AI |
| 课后整理 | 5分钟 | 整理学习心得 |

## 学习准备清单

### 材料准备
- ✓ 设备电量充足
- ✓ 网络连接正常
- ✓ AI工具账号已登录

### 心理准备
- ✓ 保持好奇心
- ✓ 大胆提问
- ✓ 耐心等待回答

### 家长准备
- ✓ 陪伴学习
- ✓ 协助操作
- ✓ 记录精彩对话

## 学习小贴士

1. 先从简单问题开始
2. 用礼貌的语言提问
3. 可以要求 ChatGPT 讲故事
4. 把有趣的回答记下来

---

让我们一起开启与AI朋友的第一段对话吧！
