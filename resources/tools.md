## 使用建议

### 第一次使用

1. 从简单的工具开始，比如 Scratch 或 Quick, Draw!
2. 家长先熟悉工具，再引导孩子使用
3. 注意屏幕时间，保护视力

### 安全注意事项

- 使用正规网站和工具
- 不要在公共网络输入个人信息
- 家长陪同使用在线工具

---

## 获取帮助

如果某个工具不会用，可以：
1. 查看工具自带的教程
2. 在网上搜索"工具名 + 教程"
3. 在 GitHub 提问
