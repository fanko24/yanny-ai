# 常见问题解答（FAQ）

> 家长最关心的问题，这里都有答案

---

## 📚 关于课程

### Q1: 这个课程适合多大孩子？

**A:** 本课程专为8岁左右的一年级小女孩设计，也适合：
- 7-9岁的小学生
- 对AI好奇但接触不多的孩子
- 想和孩子一起学习的家长

### Q2: 需要什么基础？

**A:** 不需要任何基础！
- 会基本的电脑/平板操作即可
- 会打字更好，不会可以家长帮忙
- 保持好奇心就够了

### Q3: 12节课要多久上完？

**A:** 建议每周1节，共12周（3个月）：
- 不要贪多，慢慢学
- 每节课40-60分钟
- 课后可以继续练习

### Q4: 可以跳过某些课吗？

**A:** 不建议跳课：
- 课程设计是循序渐进的
- 前面的课是后面的基础
- 如果孩子特别感兴趣某节课，可以先学，但要补前面的

## 🛠️ 关于工具

### Q5: 需要付费吗？

**A:** 大部分工具免费：
- **豆包** - 完全免费
- **文心一格** - 每天免费额度
- **ChatGPT** - 基础版免费，Plus版付费

### Q6: 用豆包还是ChatGPT？

**A:** 推荐方案：
- **首选**：豆包（中文好，国内快，免费）
- **备选**：ChatGPT（功能更强，但需要科学上网）
- **都可以**：两个都用也可以

### Q7: 需要下载APP吗？

**A:** 不一定需要：
- 豆包 - 可以用网页版，也有APP
- ChatGPT - 网页版和APP都可以
- 建议：网页版功能更全，APP更方便

### Q8: 电脑还是平板？

**A:** 都可以：
- **电脑** - 打字方便，屏幕大
- **平板** - 方便携带，触屏操作
- **手机** - 屏幕小，不太推荐

## 👨‍👩‍👧 关于学习

### Q9: 家长要全程陪同吗？

**A:** 建议分阶段：
- **前3节课** - 全程陪同
- **4-8节课** - 开始时陪同，后面可以放手
- **9-12节课** - 孩子独立完成，家长定期检查

### Q10: 孩子不感兴趣怎么办？

**A:** 几个建议：
1. 从孩子感兴趣的话题开始
2. 让孩子用AI做她喜欢的事（画画、编故事）
3. 多鼓励，少批评
4. 休息一下，过几天再试

### Q11: 孩子太依赖AI怎么办？

**A:** 要建立正确认知：
1. AI是工具，不是答案机器
2. 先自己思考，再问AI
3. 用AI来检查，不是用AI来写
4. 多做不依赖AI的练习

### Q12: 作业可以用AI做吗？

**A:** 要分情况：
- **❌ 不能**：让AI直接写作业
- **✅ 可以**：用AI解释不会的题
- **✅ 可以**：用AI检查作业
- **✅ 可以**：用AI拓展学习

## 🔒 关于安全

### Q13: AI会泄露隐私吗？

**A:** 只要注意保护，不会：
- 不要告诉AI个人信息
- 使用正规工具（豆包、ChatGPT）
- 定期清理对话记录
- 家长定期检查

### Q14: 孩子会沉迷AI吗？

**A:** 做好时间管理：
- 设置使用时间限制
- 鼓励线下活动
- 培养其他兴趣
- 家长多陪伴

### Q15: AI说错话怎么办？

**A:** AI有时会出错：
- 告诉孩子AI不是万能的
- 鼓励孩子质疑AI
- 发现错误要纠正
- 有疑问问家长或查证

### Q16: 需要开启青少年模式吗？

**A:** 强烈建议开启：
- 豆包有青少年模式
- 设置内容过滤
- 限制使用时间
- 更安全，更放心

## 🆘 遇到问题

### 技术问题

1. **打不开网站** - 检查网络，换浏览器试试
2. **登录不了** - 检查账号密码，或重新注册
3. **AI不回答** - 等一下，或换种问法
4. **回答很慢** - 网络问题，或AI在思考

### 学习问题

1. **孩子听不懂** - 家长帮忙解释，用更简单的语言
2. **作业不会做** - 参考家长指导，或跳过这题
3. **没有兴趣** - 换个角度，从孩子喜欢的开始
4. **进度太慢** - 没关系，慢慢来，质量比速度重要
