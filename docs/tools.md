## 工具使用指南

### ChatGPT使用方法

1. **注册账号**
   - 访问 https://www.doubao.com/
   - 使用手机号注册登录

2. **开始对话**
   - 在输入框输入问题
   - 点击发送或按回车

3. **绘画功能**
   - 输入"画一张..."
   - 描述想要的图片

4. **语音功能**
   - 点击麦克风图标
   - 开始语音对话

### ChatGPT使用方法

1. **注册账号**
   - 访问 https://chat.openai.com/
   - 使用邮箱注册

2. **开始对话**
   - 在输入框输入问题
   - 点击发送

3. **注意事项**
   - 需要科学上网
   - 部分功能需要付费

## 安全注意事项

1. **不要分享个人信息**
   - 不告诉AI真实姓名
   - 不说家庭住址
   - 不说电话号码

2. **家长陪伴**
   - 首次使用需要家长指导
   - 遇到问题及时询问

3. **内容安全**
   - 不问不适当的问题
   - 遇到奇怪的内容告诉家长
