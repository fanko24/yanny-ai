# Vivian AI - 8岁小女孩AI启蒙课程

> 让每个小女孩都能轻松踏入AI世界

## 项目介绍

Vivian AI 是专为8岁一年级小女孩设计的AI启蒙课程。通过有趣的故事、互动的实践和创意的项目，让孩子们在轻松愉快的氛围中了解AI、使用AI，成为小小AI专家。

## 课程特色

- 🌟 **趣味故事引导**：每节课都有生动的故事引入
- 🎨 **动手实践**：简单易懂的操作步骤
- 🎯 **渐进式学习**：从对话到创作，逐步深入
- 🏆 **成就徽章**：完成课程获得专属徽章
- 👩‍👧 **亲子互动**：家长指导手册帮助陪伴学习

## 课程结构

### 第一学期：AI魔法探险 (12周)

#### **模块一：认识AI朋友** (3周)
- 第1课：和AI聊聊天
- 第2课：AI画梦工厂
- 第3课：AI小歌手

#### **模块二：AI创作工坊** (5周)
- 第4课：小小创作家（故事创作与写作）
- 第5课：创意画室
- 第6课：配音小演员
- 第7课：音乐魔法师
- 第8课：视频魔法师

#### **模块三：AI学习伙伴** (4周)
- 第9课：作业小帮手
- 第10课：科学探险家
- 第11课：我的AI作品集
- 第12课：成果展示会

- 8岁左右的一年级小女孩
- 对AI好奇但接触不多的孩子
- 希望和孩子一起学习的家长

## 课程理念

- **寓教于乐**：通过游戏化学习激发兴趣
- **循序渐进**：从简单到复杂，适合儿童认知发展
- **创意表达**：鼓励孩子用AI表达自己的想法
- **安全第一**：所有工具都经过筛选，确保安全
