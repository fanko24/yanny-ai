# Yanny AI 课程系列

> 为8岁小朋友设计的AI入门课程，每周30-60分钟，图文并茂，动手实践

## 课程简介

这是一套专门为8岁小女孩设计的系统性人工智能学习课程。课程采用儿童友好的方式，通过有趣的故事、游戏和实践操作，让孩子轻松理解AI的基本概念，培养科学思维和创新意识。

## 课程特点

- **儿童友好**: 使用孩子能理解的语言和比喻
- **图文并茂**: 配合生动的图片和插图
- **动手实践**: 每节课都有互动实验和游戏
- **循序渐进**: 从基础概念到实际应用，逐步深入
- **家长指导**: 提供详细的授课要点和建议

## 使用指南

### 准备工作

1. 确保有稳定的网络连接（用于在线体验AI工具）
2. 准备一台电脑或平板设备
3. 准备纸笔、画笔等文具
4. 预留30-60分钟的授课时间

### 授课流程

1. **课前准备**: 阅读 `parent-guide.md`，准备所需材料
2. **正式授课**: 按照 `content.md` 的顺序进行
3. **动手实践**: 引导孩子完成 `hands-on.md` 中的活动
4. **课后作业**: 布置 `homework.md` 中的任务
5. **进度记录**: 更新 `docs/progress.md` 中的学习进度

### 时间建议

- 每周1节课
- 每节课30-60分钟（根据孩子注意力调整）
- 可选择周末进行，避免影响正常学习

## 授课建议

1. **保持耐心**: 每个孩子的学习节奏不同，不要急于求成
2. **多鼓励**: 对孩子的每一个发现和想法给予肯定
3. **灵活调整**: 根据孩子的兴趣和反应调整授课内容和节奏
4. **联系生活**: 将AI知识与日常生活联系起来，增加实用性
5. **记录成长**: 保存孩子的作品和笔记，见证进步
